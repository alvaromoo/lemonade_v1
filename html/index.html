Bienvenido a Lemonade

Lemonade es una estructura de proyecto inicial para desarrollo Frontend. El objetivo es facilitar el inicio de nuevos proyectos, ofreciéndote una base sólida, potente y flexible sobre la que poder hacer que tus ideas se hagan realidad.

Lemonade utiliza dos potentes preprocesadores que lo harán todo más sencillo:

  • JADE
  • SASS

Espero que el proyecto te sea de utilidad.

¡Un abrazo!
