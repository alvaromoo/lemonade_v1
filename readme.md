# [Lemonade](http://alvaromoo.com)
----------------------------------
Lemonade es un entorno de desarrollo enfocado a Front-end.

El objetivo del proyecto es convertirse en un punto de partida para desarrollo front de webs/apps de manera sencilla, ligera y rápida. Dada su rápida implementación, también sirve como entorno de pruebas para desarrolladores Front-end, tanto para practicar y evolucionar sus habilidades como desarrolladores como en la realización de pruebas para desarrollos de sus trabajos.
